# Клиентское приложение - Туристическое агентство

React приложение для бронирования туров.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу http://localhost:3001

## Функциональность

- **Авторизация** - вход и регистрация пользователей
- **Просмотр туров** - список туров с фильтрацией
- **Бронирование** - создание заказов на туры
- **Мои заказы** - просмотр и отмена своих заказов
- **Панель управления** (для модераторов/админов):
  - Управление турами
  - Управление маршрутами
  - Управление группами заездов
  - Управление клиентами
  - Управление заказами
  - Управление пользователями (только для админов)

## API

Клиент использует прокси для API запросов к серверу (localhost:3002).
